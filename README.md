# ProjectAddons
Side plugin for various addon abilities to ProjectKorra plugin

## Abilities
### Avatar
- EnergyBeam

### Chiblocking
- Jab
- NinjaStance
- ChiblockingJab (Combo)
- FlyingKick (Combo)
- WeakeningJab (Combo)

### Earth
- EarthKick
- LavaFlux
- LavaSurge
- MetalRepair
- Shrapnel

### Fire
- Explode
- FireDisc
- FlameBreath (Combo)
